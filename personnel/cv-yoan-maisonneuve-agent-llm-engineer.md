## Selected Work — IA agentic & cognition étendue (2025-2026)

### **matchllm V0.2 — Système agentic theory-grounded** · *avril 2026*
Framework agentic Python avec **12 champs directionnels** (DIR/STR/GRA/META/OPS/ARC/STY/ETH/CTX/EXP/ROB/PED) pondérés λ, **6 moteurs cognitifs** (Invariant / Field / Trajectory / Evaluation / Correction / Archive), **5 régulateurs dynamiques** (Corridor / Entropy / Attractor / Phase / Homeostasis).
- 7 100 lignes Python · 2 500 lignes documentation · 49/49 tests pytest verts
- **Benchmark contrôlé live (Claude Sonnet 4.5) : 4/4 wins vs prompt brut · +24 % global · +181 % OPS**
- LLM-judge fonctionnel (Sonnet génère, Haiku évalue avec critique actionnable)
- Clients pluggables (Mock / Anthropic / OpenAI), API REST FastAPI, package `askio1`

### **Cyborg — Framework de cognition externalisée** · *avril 2026*
Architecture en **4 organes externalisés** (MODELE / PERCEPTION / PREDICTION / ERREUR) implémentant le cycle d'inférence active de Friston : `MODELE → PREDICTION → PERCEPTION → ERREUR → ajuste MODELE`.
- Install scriptée 2 secondes (`cyborg-install.ps1`) · hook vocal `SessionStart` (TTS PowerShell)
- Vocabulaire scientifique strict (Clark 1998–2024, Friston 2010, Kauffman 2000)
- Template transférable ; instances dérivées : CyborgV0.1, cyborg-yoan

### **CyborgV0.1 — Bras robotique 2D apprenant par inférence active** · *avril 2026*
Implémentation Python+numpy d'un bras 2 articulations qui apprend à atteindre une cible par minimisation de l'erreur de prédiction. Premier livrable empirique de l'équivalence Friston / contrôle moteur prédictif.
- CLI vivante avec rendu ASCII · erreur baisse mesurée (5 cycles, seed=42 reproductible)

### **cyborg-yoan — Mémoire externe portable pour LLM** · *avril 2026*
Système d'externalisation cognitive injectable dans n'importe quel projet par n'importe quelle IA (`yoan.md`). Bootstrap interactif, journalisation structurée.

### **Blueprint — Hiérarchie logique L0 → L4** · *2025–2026*
Synthèse théorique unifiée pour orchestration multi-agents. 5 couches (matière / forme / cohérence / règle / direction), 5 opérateurs canoniques (Match / Cluster / Select / Reason / Meta), équipe canonique (Chef / Observateur / Scribe / Correcteur). Règle multi-agents : L0–L1 locales, L2–L4 synchronisées.
